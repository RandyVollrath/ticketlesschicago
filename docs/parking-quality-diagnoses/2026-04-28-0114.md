## Proposed fix for future runs

For the autonomous improver to function in a sandboxed environment, one of
the following must be true:

1. **Inject env vars at agent-launch time.** Pass `NEXT_PUBLIC_SUPABASE_URL`
   and `SUPABASE_SERVICE_ROLE_KEY` as environment variables when spawning the
   agent so the local script can connect without network egress.

2. **Export a static report artifact.** Have the daily cron write the
   `DiagnosisReport` JSON to a file in the repo (e.g.
   `data/parking-quality-latest.json`, gitignored or committed temporarily)
   before the improver runs. The improver reads the file instead of calling
   any network endpoint.

3. **Allowlist the endpoint in the Claude Code sandbox config.** Add
   `autopilotamerica.com` to the network allowlist so the Bearer-authenticated
   endpoint is reachable.
