## What is needed to fix this

**Option A — add the external URL to the Bash allowlist:**
In `.claude/settings.json`, add `autopilotamerica.com` to the allowedHosts (or
equivalent) list for Bash network calls. This lets the agent curl the live
admin endpoint with the `CRON_SECRET` bearer token.

**Option B — provide a `.env.local`:**
A `.env.local` with `NEXT_PUBLIC_SUPABASE_URL` and `SUPABASE_SERVICE_ROLE_KEY`
allows the local `scripts/parking-quality-diagnose.ts` script to run directly
against Supabase without any outbound HTTP call. This is the more robust option
because it does not require the production service to be up.

Both options can coexist.
